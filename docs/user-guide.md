# User Guide

## 基本操作

1. `画像を追加` で画像ファイルを選択します。対応ブラウザでは `フォルダ読込` でフォルダ内の画像をまとめて読み込めます。
2. 左または上部の画像一覧から編集する画像を選びます。
3. 必要な設定を選び、キャンバス上をドラッグしてモザイクを適用します。
4. `元に戻す`、`この画像をリセット`、`Before/After確認` で結果を確認します。
5. `一括保存` で加工結果を ZIP として保存します。

## プリセット

- `Fantia ピクセル`: 添付参考のピクセル化例に合わせ、矩形選択、ピクセル化、強めのブロックサイズを適用します。
- `Fantia ぼかし`: 添付参考のぼかし例に合わせ、矩形選択、ぼかし、強めのぼかし幅を適用します。
- `Skeb 1%`: 読み込み済み画像の長辺を基準に、ブロックサイズを `ceil(長辺 * 0.01)` 以上にします。画像がない時は選択できません。

プリセットは設定値を切り替えるだけで、画像データや既存の編集履歴は削除しません。

## プレビュー操作

- `幅に合わせる`: プレビュー領域の幅に合わせます。
- `高さに合わせる`: プレビュー領域の高さに合わせます。
- `100%表示`: 元画像のピクセル相当で表示します。
- `拡大` / `縮小`: 任意倍率に切り替えて表示倍率を調整します。
- `パン`: 拡大中にドラッグで表示位置を移動します。もう一度押すと編集に戻ります。

表示倍率とパンは見た目だけの操作です。書き出し画像のピクセル数やモザイク処理内容は変わりません。

## モバイル操作

- 画像一覧から画像を選ぶと、一覧は自動で折りたたまれます。
- 一覧をもう一度表示したい場合は、目のアイコンを押します。
- フォルダ読み込みや ZIP 保存の挙動は、モバイルブラウザごとのファイル機能に依存します。

## 手動確認チェック

- アプリが日本語 UI で表示される。
- 画像追加、フォルダ読み込み、一括保存のボタンが使える。
- Fantia/Skeb プリセットを押すと設定値が切り替わる。
- ブラシまたは矩形のドラッグ中に範囲表示が出る。
- Before/After ボタンで加工前後を切り替えられる。
- 100% 表示、拡大、縮小、パンでプレビュー位置を調整できる。
- モバイル幅で画像を選ぶと一覧が折りたたまれ、再表示もできる。
- ZIP 保存後に画像一覧がリセットされる。
